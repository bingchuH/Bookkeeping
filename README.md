# Bookkeeping · 记账 Skill

<p align="center">
  <b>一个为 AI Agent 设计的本地记账 Skill</b>
</p>

Bookkeeping 是一个为 **AI Agent** 设计的本地记账 Skill。  

它能让你的agent自己分析文件结构、识别交易类型、排除内部流转、映射账户、检测重复，并最终把干净的账目写入你的本地账簿。

无论你用的是 Claude Code、Codex还是其他同类 AI Agent，都可以用一句话完成记账、导入账单、处理借贷、查看财务看板——所有数据都安静地留在你自己的电脑里。

## 核心能力一览

✨ **记账与自动分类**

支持 `支出`、`收入`、`转账`、`借入`、`借出`、`还款`、`收款`、`余额修正` 八种资金动作。  

📥 **支付宝 & 微信 & 多种银行账单导入**

直接把导出的表格路径丢进对话，让它帮你记账。Skill 会自动为你完成：

- 自动识别退款、红包、二手交易等特殊类型；
- 自动跳过关闭的交易和余额宝/零钱内部流转；
- 根据交易对象智能推荐分类（如 `12306` → 交通/火车高铁）；
- 缺少信息的记录交由人工确认，避免错误入账。

🔁 **智能去重**

无论是手工记过又导入同一笔，还是支付宝和微信重复记录了同一笔支付——  
Skill 会精准判定重复，防止资产虚增。  
尤其针对「银行卡扣款」和「支付平台账单」重复记录，它会结合账户别名映射准确去重，不再让你手动比对。

🧠 **关键词规则学习**

你可以教它：  
“以后所有交易对方包含‘饿了么’的支出，都归到 `餐饮/早午晚餐`。”  
Skill 会记住这条规则，并优先用更精确、更长的关键词进行匹配。  
它还会帮你分析还没分类的高频词，越来越好用。

📊 **财务看板（Dashboard）**

更多功能自己探索吧

## 初心

我已经记账八年了。八年里，几乎每笔开销都是手动维护，低效、枯燥。
我一直想要一个工具：**能自动导入支付宝、微信和银行卡账单，还能聪明地识别出「银行卡扣款」和「支付宝消费」其实是同一笔钱，自动去重。**  

很可惜，市面上我试过的 App ，虽然可以导入网络账单，但要么无法跨账单来源去重，要么把数据同步到云端让我不安，导出数据还需要开会员。

直到 AI Agent 的进步，我意识到：**这些脏活、累活，完全可以交给 Agent 来完成**  

于是就有了这个项目——一个帮你自动完成清洗、去重、分类与统计的记账 Skill。    

希望它能让你从「记账的疲惫」里解脱出来，把精力留给真正值得用心的事。
